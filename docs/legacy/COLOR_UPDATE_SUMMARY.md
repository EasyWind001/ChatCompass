- **从**: 纯黑灰色调 (GitHub Dark)
- **到**: 深蓝灰色调 (更柔和现代)

```

```


## 🎓 设计理念

### 深蓝灰基调的选择
- **蓝色**: 代表科技感、专业、冷静
- **灰色**: 中性、现代、稳重
- **适中亮度**: 兼顾护眼和可见度

### 13% 亮度的黄金值
- 比 7% 明显更亮,视觉舒适
- 比 20% 更护眼,适合长时间使用
- 完美平衡亮度和深色主题特性
